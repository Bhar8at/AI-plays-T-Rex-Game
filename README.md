# AI-plays-T-Rex-Game

- Actions
    - Do nothing = [1,0]
    - Jump = [0,1]

- Observations describing a state 
    - game_over status
    - incoming cactus position
    - incoming cactus speed
    - score
   

- Reward
    - Jump over cactus = +10
    - Touch cactus = -10
    - Do nothing = keep running +5
